Cómo cargar un producto nuevo al catálogo
📂
Abrir el archivo compra-directa.html
Abrilo con el Bloc de Notas (Windows) o cualquier editor de texto. Click derecho → Abrir con → Bloc de notas.
🔍
Buscar el lugar donde agregar
Presioná Ctrl+F y buscá este texto exacto:
AGREGAR PRODUCTOS AQUI
Ahí es donde pegás el nuevo producto.
📋
Copiar este bloque y completar los datos
Copiá el bloque de abajo, pegalo antes del comentario y completá cada campo:
  /* Pegá esto antes de: AGREGAR PRODUCTOS AQUI */
  {
    id: 99,              ← numero unico, no repetir
    cat: "Celulares",    ← Celulares / Smart TV / Audio / Accesorios / Hogar
    name: "Nombre",     ← nombre del producto
    sub: "Specs",      ← specs cortos ej: 128GB · 6.1" · 5G
    desc: "Detalle",   ← descripcion breve
    img: "URL foto",   ← URL de imagen o nombre del archivo
    badge: "Nuevo",    ← Nuevo / Oferta / Top ventas / null (sin badge)
    badgeBg: "#0066FF",← color: #0066FF=azul #FF6900=naranja #00A651=verde
    icon: "📱",       ← emoji si no carga la foto
    ars: "980.000"   ← precio en pesos. null = muestra "Consultar"
  },
💡
Cuando el precio aun no esta definido pone ars: null — el catalogo muestra "Consultar" y el cliente va directo a WhatsApp.
🎨
Colores de badge disponibles
Top ventas Nuevo Oferta Premium Destacado Promo
💾
Guardar y listo
Guardá el archivo con Ctrl+S y abrilo en Chrome. El producto aparece automáticamente en el catálogo.
2
Cómo usar la calculadora de precios
1️⃣
El dólar se carga automático
Al abrir en Chrome trae la cotización del momento. Si no hay internet, escribís el valor vos manualmente en el campo.
2️⃣
Ingresás el costo en USD
El precio que te da el proveedor en dólares.
3️⃣
Seleccionás categoría y tipo
Celular slim (sin cargador): 8% · + U$D 6 si es menor a U$D 300
Celular gruesa (con cargador): 10% · + U$D 4 si es menor a U$D 300
Electro chico (caja zapatilla): 20% todo incluido
Electro grande (robot vacuum+): 25% todo incluido
4️⃣
El precio final aparece solo
Ves el desglose completo y el precio final redondeado listo para copiar con un toque y cargarlo al catálogo.
Ejemplo · JBL Go 4 · U$D 30.50 · Dólar $1.405
Costo productoU$D 30.50
× Cotización dólar$ 1.405
Subtotal en pesos$ 42.852
× 1.20 (electro chico)+ $ 8.570
Precio final$ 51.500
3
Planilla de registro de clientes
👥
¿Para qué sirve?
Cada cliente que compra o consulta queda registrado. Con eso armás la comunidad — podés avisarles de nuevos productos, ofertas y novedades por WhatsApp.
📋
Datos a registrar por cliente
• Nombre y apellido
• WhatsApp
• Zona / Localidad
• Tipo de cliente (particular / comercio / revendedor)
• Producto que compró o consultó
• Fecha
• Estado (consulta / compra confirmada / entregado)
🚀
Con 50 clientes registrados ya tenés una lista de difusion por WhatsApp. Cada vez que entra un producto nuevo, un mensaje y ya tenés ventas antes de publicar en el catalogo.
📊
¿Querés la planilla en Excel o en Google Sheets?
La armamos ahora con todos los campos, filtros por zona y tipo de cliente, y un dashboard simple de ventas. Decime cuál preferís.
